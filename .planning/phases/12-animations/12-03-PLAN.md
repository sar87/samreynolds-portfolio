1. Update card hover to lift more (-6px to -8px) with enhanced shadow:
   ```css
   .card {
     /* Existing styles... */

     /* Prepare for GPU-accelerated hover */
     will-change: transform;
   }

   @media (prefers-reduced-motion: no-preference) {
     .card {
       transition:
         border-color var(--animation-duration-fast, 150ms) var(--animation-easing-out),
         box-shadow var(--animation-duration-fast, 150ms) var(--animation-easing-out),
         transform var(--animation-duration-fast, 150ms) var(--animation-easing-out);
     }

     .card:hover {
       transform: translateY(-6px);
       box-shadow: var(--shadow-lg);
       border-color: var(--color-accent);
     }
   }
   ```

2. Ensure will-change is removed for reduced-motion users:
   ```css
   @media (prefers-reduced-motion: reduce) {
     .card {
       will-change: auto;
     }

     .card:hover {
       /* Keep border color change for feedback, but no movement */
       border-color: var(--color-accent);
       box-shadow: var(--shadow-md);
       transform: none;
     }
   }
   ```

The current card hover only lifts -2px and uses shadow-md. This update increases lift to -6px and uses shadow-lg for more pronounced feedback per user request.
