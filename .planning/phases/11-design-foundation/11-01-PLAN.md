  <action>
Add layered shadow tokens after the border radius section:

```css
/* Shadow elevation - layered for realistic depth */
--shadow-sm:
  0 1px 2px rgba(0, 0, 0, 0.04),
  0 1px 1px rgba(0, 0, 0, 0.03);

--shadow-md:
  0 2px 4px rgba(0, 0, 0, 0.04),
  0 4px 8px rgba(0, 0, 0, 0.04),
  0 8px 16px rgba(0, 0, 0, 0.02);

--shadow-lg:
  0 4px 8px rgba(0, 0, 0, 0.04),
  0 8px 16px rgba(0, 0, 0, 0.04),
  0 16px 32px rgba(0, 0, 0, 0.02);
```

These follow Josh Comeau's layered shadow pattern from RESEARCH.md - multiple subtle layers create more realistic depth than a single shadow.
