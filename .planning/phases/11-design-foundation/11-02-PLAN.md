<task type="auto">
  <name>Task 2: Add halftone texture overlay to global.css</name>
  <files>src/styles/global.css</files>
  <action>
Add the full-page texture overlay using body::after pseudo-element.

After the body styles section, add:

```css
/* Halftone texture overlay - newspaper/print aesthetic */
body::after {
  content: '';
  position: fixed;
  inset: 0;
  background-image: url('/assets/textures/halftone-dots.png');
  background-size: var(--texture-size, 180px);
  background-repeat: repeat;
  mix-blend-mode: multiply;
  pointer-events: none;  /* Critical: allows clicking through */
  z-index: 9999;
  opacity: var(--texture-opacity, 0.12);
}
```

CRITICAL: The `pointer-events: none` is essential - without it, nothing on the page will be clickable.

Note: The path /assets/textures/ assumes Vite serves from public/. If this doesn't work, adjust to the correct asset path.
  </action>

  </verify>

<task type="auto">
  <name>Task 3: Add comprehensive reduced-motion handling</name>
  <files>src/styles/global.css</files>
  <action>
Add reduced-motion media query at the end of global.css:

```css
/* Reduced motion - respect accessibility preferences */
@media (prefers-reduced-motion: reduce) {
  *,
  *::before,
  *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
    scroll-behavior: auto !important;
  }
}
```

This is the nuclear option that disables ALL motion site-wide when user prefers reduced motion. It's aggressive but ensures WCAG compliance.

Also update the existing scroll-behavior rule to use the no-preference pattern:
```css
@media (prefers-reduced-motion: no-preference) {
  html {
    scroll-behavior: smooth;
  }
}
```

Remove the existing separate reduced-motion media query for scroll-behavior if present (consolidate into the new comprehensive rule).
  </action>
  <verify>

Expected: At least 2 (no-preference and reduce)
  </verify>
  <done>Comprehensive reduced-motion handling implemented for accessibility</done>
