  <action>
Update the .card class to use layered shadows:

1. Replace the current border-only approach with shadow + border:
```css
.card {
  padding: var(--space-6);
  background: var(--color-bg-elevated, var(--color-white));
  border: 1px solid var(--color-border);
  border-radius: var(--radius-md);
  box-shadow: var(--shadow-sm);
}
```

2. Update hover state:
```css
.card:hover {
  border-color: var(--color-accent);
  box-shadow: var(--shadow-md);
}
```

3. Add transitions only for users without motion preference. Wrap existing transition in media query:
```css
@media (prefers-reduced-motion: no-preference) {
  .card {
    transition:
      border-color var(--transition-fast),
      box-shadow var(--transition-fast),
      transform var(--transition-fast);
  }

  .card:hover {
    transform: translateY(-2px);
  }
}
```

This gives cards a subtle lift effect on hover (shadow + slight upward movement).
